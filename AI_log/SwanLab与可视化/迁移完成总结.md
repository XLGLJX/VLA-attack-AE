## 📋 需要提供的信息

### 1. SwanLab API Key（必需）

**获取方式：**
1. 访问 https://swanlab.cn
2. 注册/登录账号
3. 点击右上角头像 → 设置 → API Keys
4. 复制 API Key

**配置方式：**

```bash
# 方式一：命令行登录
swanlab login
# 然后输入你的 API Key

# 方式二：环境变量
export SWANLAB_API_KEY="your_api_key_here"
```

### 2. 项目名称（可选）

- 默认项目名：`VLA-Attack`
- 可通过 `--swanlab_project` 参数自定义
