### 6. 安装flash-attn
```bash
# 首先降级NumPy以解决兼容性问题
pip install "numpy<2"

# 清理pip缓存
pip cache purge

# 手动下载预编译wheel
cd /tmp
wget https://github.com/Dao-AILab/flash-attention/releases/download/v2.5.5/flash_attn-2.5.5+cu122torch2.2cxx11abiFALSE-cp310-cp310-linux_x86_64.whl

# 安装下载的wheel文件
pip install /tmp/flash_attn-2.5.5+cu122torch2.2cxx11abiFALSE-cp310-cp310-linux_x86_64.whl
```
**状态**: ✅ 成功

**遇到的问题及解决**:
1. **NumPy 2.x兼容性问题**:
   - 错误: "Failed to initialize NumPy: _ARRAY_API not found"
   - 解决: 降级NumPy从2.2.5到1.26.4

2. **跨设备链接错误**:
   - 错误: "[Errno 18] Invalid cross-device link"
   - 原因: pip缓存目录与目标目录在不同文件系统
   - 解决: 清理pip缓存 (`pip cache purge`)

3. **网络连接问题**:
   - 问题: pip自动下载时连接超时
   - 解决: 使用wget手动下载预编译wheel文件（115MB，耗时10分8秒）

**最终状态**: flash-attn 2.5.5 安装成功

## 问题总结

### 已解决的问题

1. **PyTorch版本冲突**
   - 问题: conda安装的PyTorch 2.5.1与项目要求的2.2.0不匹配
   - 解决: pip自动降级到正确版本

2. **NumPy版本兼容性**
   - 问题: NumPy 2.2.5与flash-attn不兼容
   - 解决: 降级到NumPy 1.26.4

3. **flash-attn安装失败**
   - 问题1: 跨设备链接错误
   - 解决: 清理pip缓存 (`pip cache purge`)
   - 问题2: 网络连接超时
   - 解决: 使用wget手动下载预编译wheel文件后本地安装
