## 下载方式

### 方案一：使用 Git LFS（推荐）

```bash
# 1. 进入数据集目录
cd /mnt/home/lvmingyuan/marionette/VLA-attack-AE/dataset

# 2. 检查是否安装 git-lfs
git lfs version

# 3. 如果未安装，先安装 git-lfs
# Ubuntu/Debian:
sudo apt-get install git-lfs

# 4. 初始化 git-lfs
git lfs install

# 5. 下载所有 LFS 文件
git lfs pull

# 或者只下载特定数据集
git lfs pull --include="libero_spatial_no_noops/**"
```

### 方案二：从源仓库重新克隆

如果数据集是从 Git 仓库克隆的：

```bash
# 1. 备份当前目录
cd /mnt/home/lvmingyuan/marionette/VLA-attack-AE
mv dataset dataset_backup

# 2. 重新克隆（带 LFS）
git lfs clone <数据集仓库URL> dataset

# 或者
git clone <数据集仓库URL> dataset
cd dataset
git lfs pull
```

### 方案三：手动下载

如果有直接下载链接，可以手动下载 tfrecord 文件并替换指针文件。
